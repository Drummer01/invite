# 💍 Запрошення на весілля - Світлана та Андрій

Красивий односторінковий веб-сайт запрошення на весілля, створений за допомогою React, Vite і сучасних веб-технологій.

## ✨ Особливості

- **Современна типографія**: Використовуються шрифти `Playfair Display` та `Lora` для класичного дизайну
- **Нейтральна кольорова палітра**: Теплі бежеві та нюдові тони
- **Гладкі анімації**: Переходи та вхідні анімації для всіх елементів
- **Адаптивний дизайн**: Прекрасно виглядає на всіх пристроях
- **Локалізовано**: Повністю на українській мові

## 📋 Структура сторінки

1. **Hero-секція** - Привітання з датою та фото пари
2. **Календар** - Серпень 2026 з виділеною датою весілля (16 серпня)
3. **Розклад дня** - Часова線 з основними подіями
4. **Місце проведення** - Адреса та локація Forest House
5. **Палітра для гостей** - Рекомендовані колори для одягу
6. **Підпис** - Завершуюче послання від пари

## 🚀 Запуск проекту

### Встановлення залежностей
```bash
npm install
```

### Розробка
```bash
npm run dev
```
Сайт буде доступний на `http://localhost:5173/`

###빌д для виробництва
```bash
npm run build
```

### Передпросмотр продакшену
```bash
npm run preview
```

## 📝 Налаштування

Щоб налаштувати сайт під себе, відредагуйте:

- Фото пари в `src/components/Hero.jsx`
- Дату в `src/components/Calendar.jsx`
- Адресу в `src/components/Location.jsx`
- Колори в `src/components/DressCode.jsx`
- Тексти у відповідних компонентах
